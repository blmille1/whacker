## Agent skills

### Issue tracker

Issues are tracked as GitHub issues in `blmille1/whacker` (via `gh` CLI). External PRs are **not** a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage.md`.

### Domain docs

Single-context: `CONTEXT.md` and `docs/adr/` at the repo root. See `docs/agents/domain.md`.

### Other
If you install/upgrade an npm package, verify with `npm i`. If the package had patches, delete them.
